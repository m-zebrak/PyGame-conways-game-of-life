Conway's Game of Life
=====================

My PyGame implementation of Conway's Game of Life.

Running
-------
Install PyGame using:

```` pip install pygame```` 

Then simply run the:

```` gameoflife.bat```` 

Controls
--------

- p - to toggle start/stop (pause)
- r - to randomize the grid
- n - display next generation
- c - clear grid
- q - to quit the game
